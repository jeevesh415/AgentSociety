# Memory

When selected, this skill batch-persists buffered internal states generated by cognition (need adjustments, emotions, thoughts) to long-term memory.

## Why this skill exists

The `cognition` skill (priority=40) writes multiple records to `agent._cognition_memory` during execution. This skill flushes those records to long-term memory when it runs.

## What It Does

1. **Flush cognition memory** — Batch-commit buffered entries from `_cognition_memory` grouped by type to long-term memory (pure memory operation, 0 LLM calls).
2. **Intention query** (optional) — Query current intention every 2 steps (1 LLM call), triggered only when cognition skill has run.

## Key Methods on Agent

| Method | Purpose |
|--------|---------|
| `_flush_cognition_memory_to_memory()` | Batch-commit cognition scratchpad to long-term |
| `_query_current_intention()` | LLM query for current intention (optional) |
